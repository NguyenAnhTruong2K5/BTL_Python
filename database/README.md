NGÔN NGỮ: SQL SERVER

Thứ tự SQL: schema → procedures → triggers → seed_data

schema: chứa bảng (Table)
procedures: chứa các stored procedure
triggers: chứa các trigger
seed_data: chứa các dữ liệu mẫu (như giá, bãi xe có sẵn)
